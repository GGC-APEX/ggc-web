## Formato de cada carousel

Cada carpeta contiene un `slides.md` con:
- **Specs de diseño** (paleta, tipografía, ratio)
- **Contenido slide a slide** (titular, visual, copy)
- **Caption** listo para publicar en Instagram
- **Hashtags** relevantes

## Especificaciones de diseño

- **Paleta:** Fondo #0A0A0A, texto #FFFFFF, acentos #D4A017 (dorado/ámbar)
- **Ratio:** 4:5 (1080×1350px) preferido, 1:1 (1080×1080px) también válido
- **Tipografía:** Inter Bold / Helvetica Neue Bold — transmite autoridad
- **Cuenta:** @sir.fortuna
